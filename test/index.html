Dropdown Extractor

Dropdown Extractor is a Chrome extension that copies items from dropdown lists into your clipboard.

  • Chrome Web Store: Dropdown Extractor (pending review)
  • Github project: Dropdown Extractor

Dropdown Playground

This is a testing page for Dropdown Extractor extension.
Here you can configure a single dropdown to test extraction behavior.

How?
  1. Configure test dropdown using the controls below.
    • Choose the type of dropdown from the "Type" selector.
    • Set the number of items to generate.
    • Optionally, edit item text/value/data-value in the table below.
    • Buttons in the table header can clear or toggle missing values for each column.
    • "Fill with random values" button populates the table with random data.
  2. Select the type of dropdown.
  3. Set the number of items and their text/value/data-value.
  4. Click "Render dropdown" to build the dropdown.
  5. Use the extension to extract items from the rendered dropdown.
